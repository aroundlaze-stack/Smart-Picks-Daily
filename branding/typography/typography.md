# Smart Picks Daily — Typography

## Primary Font: Space Grotesk
**Google Fonts:** `@import url('https://fonts.googleapis.com/css2?family=Space+Grotesk:wght@400;500;600;700&display=swap');`

**Why:** Modern geometric sans-serif with distinctive character. Excellent for headings and brand identity. The clean letterforms read well at all sizes and pair beautifully with Inter for body text. Used by Notion, Linear, and other premium tech brands.

## Secondary Font: Inter
**Google Fonts:** `@import url('https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap');`

**Why:** The gold standard for UI text. Optimized for screen readability. Neutral and professional. Balances the personality of Space Grotesk in headings.

## Usage
| Role | Font | Weight | Size |
|------|------|--------|------|
| H1 | Space Grotesk | 700 | 3.5-5rem |
| H2 | Space Grotesk | 700 | 2-3rem |
| H3 | Space Grotesk | 600 | 1.5-2rem |
| Navigation | Space Grotesk / Inter | 500 | 14px |
| Body | Inter | 400 | 16px |
| Captions | Inter | 400 | 13px |
| Buttons | Inter | 600 | 14-16px |
